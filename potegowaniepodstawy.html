Potęgowanie jest to działanie matematyczne wprowadzone po to, aby ułatwić zapis wielokrotnego mnożenia.
n - wykładnik potęgi
a - podstawa potęgi
b - wynik potęgowania
an = a * a * a - n razy
an = b

Przykłady:

32 = 3 * 3 = 9
33 = 3 * 3 * 3 = 27
34 = 3 * 3 * 3 * 3 = 81
Potęgowanie jest działaniem odwrotnym do pierwiastkowania.
